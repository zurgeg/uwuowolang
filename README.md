# uwuowolang
a weird lang

## commands
`u`: changes `w` to `u`

`o`: changes `w` to `u`

`w`: prints `w`

`uw`: subtract `u` from `w`

`ow`: subtract `o` from `w`

`wu`: add `w` to `u`

`wo`: add `w` to `o`

`uo`: set `u` to `o`

`ou`: set `o` to `u`

`uwu`: multiply `u` by 2

`owo`: multiply `o` by 2

`uwo`: multiply `u` by `o`

`0_0`: end command
